The ``1-my_list`` module
=========================

Import the class under test
-----------------------------
::

    >>> MyList = __import__('1-my_list').MyList

MyList is a list
------------------

    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True

Basic sorting
--------------

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Already sorted list
----------------------

    >>> sorted_list = MyList([1, 2, 3])
    >>> sorted_list.print_sorted()
    [1, 2, 3]

Negative numbers
------------------

    >>> neg_list = MyList([3, -1, 2, -5, 0])
    >>> neg_list.print_sorted()
    [-5, -1, 0, 2, 3]

Empty list
------------

    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Single element
----------------

    >>> single = MyList([42])
    >>> single.print_sorted()
    [42]

Duplicate elements
--------------------

    >>> dup_list = MyList([5, 3, 5, 1, 3])
    >>> dup_list.print_sorted()
    [1, 3, 3, 5, 5]

Original list is unchanged
-----------------------------

    >>> my_list = MyList([9, 1, 8, 2])
    >>> my_list.print_sorted()
    [1, 2, 8, 9]
    >>> my_list
    [9, 1, 8, 2]

print_sorted returns None
----------------------------

    >>> result = MyList([2, 1]).print_sorted()
    [1, 2]
    >>> print(result)
    None
